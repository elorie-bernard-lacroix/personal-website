Automated water sanitation system

Course Project @ University of Toronto (ESC204 Praxis III). With Jessica M., Hyunji K., Preetinder S., Kristen M., Seoyun Y., Kumkum C., and Joaquin C.

With the use of 3D printing, sensors, motors, and online tools such as Fusion360 and Notion, my team and I prototyped a system that could heat water to a specified temperature, for a given time, and then automatically dispense it to a tank of potable water. A presentation page for this project is coming soon!

Seam-carving program

Course Project @ University of Toronto (ESC190 Data Structures & Algorithms). With Chaewon L.

This C program uses dynamic programming to identify vertical paths of pixels in an image that can be removed without distorting the image. It can continue to remove these less important paths one by one, gradually making the image less wide without distorting it or removing meaningful aspects of it. For more details, please refer to the GitHub repository.

MATLAB Script that predicts the failure load of a specified bridge

Final Project @ University of Toronto (CIV102 Structures and Materials). With Christine L. and Jocelyn Q.

This MATLAB script calculates the factor of safety for various bridge designs given a load of 400 N. It creates an envelope shear force and considers various types of failure including plate buckling, flexural compression, shear force in the glue and in the MATboard of the bridge, etc. This code was used to build an 850 g bridge out of matboard that supported a live load of 1000 N (100 times greater than its self-weight). This result ranked us third place in the Engineering Science 2T6 class. For more details, please refer to its GitHub repository and the engineering drawings.

Synonym Detector
